To install Joomla and JOpensim, follow these steps:

  1. Download the zip file from https://www.joomla.org/.
  2. Extract it to Outworldzfiles\Apache\htdocs\JOpensim folder.
  3. Go to Setup->Settings->Apache Settings and set the type to "JOpensim".
  4. Click the Apache Icon at top right of the main panel and select Restart.
  5. Navigate to "http://localhost" and follow the instructions to set up  Joomla.
  6. Go to https://www.jopensim.com/jopensim/downloads1.html and follow the instructions to set up JOpensim.
