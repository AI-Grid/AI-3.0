To install Wordpress, follow these steps:

  1. Download the zip file from https://wordpress.org/download/.
  2. Extract it to Outworldzfiles\Apache\htdocs\Wordpress folder.
  3. Go to Setup->Settings->Apache Settings and set the type to "Wordpress".
  4. Click the Apache Icon at top right of the main panel and select Restart.
  5. Navigate to "http://localhost" and follow the instructions to set up  Wordpress.
