Backup - Autobackup

if enabled, the Autobackup module periodically saves data to a folder.    

If Auto Backup is enabled, Opensim will make an OAR backup of each region after Opensim has run for Interval time.  If there are checkboxes set in the System Backup, then those files will be backup up, too. The files will appear in the Outworldzfiles\Autobackup folder. 

Keep for Days will delete any OAR or  backup zip older than this period of time.  

Save To Folder: You can click on “Autobackup” folder name, or the folder Icon, and set a different location.

Backup:

You can get Help on Backing up all Data files and how to Export SQL files in the top menu.
