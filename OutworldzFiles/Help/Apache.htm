Apache Webserver

Apache is a powerful and flexible Web Server. It is part is a WAMP stack (Windows Apache Mysql and PHP) that I specially configured for Opensim and DreamGrid.  You do not need to enable it.

Apache can serve streaming movies, MP3 audio and other content to Opensim when enabled.   You can enable Word Press, Joomla/jOpensim and any other Content Management Systems (CMS) by enabling Apache.  Apache will always run, even if you turn off your grid offline, so your Web Site, Blog, or other content can always be ready to go.

Web Port

The WWW Default port is 80. If your ISP blocks port 80, try 8000. You will need to Port Forward this port in your Routers firewall for others to see it.   Once Apache is running, you can access the web server at http://127.0.0.1 or http://127.0.0.1:80.  If you changed the port, change the number on the end.

When enabled and Opensim is running a map will appear at http://127.0.0.1. You can hover on any square and see its size, location, name, and other details. 

Enable Apache Web Server

The checkbox for [Enable Apache Web Server] must be set so the web server will install and start itself.  You will need to configure Maps and the starting location in the Setup->Settings Map setup screen.

Apache will then start when your PC is booted as it will install and run as a service. But if you run a DreamGrid copy from another folder, the server will still attempt to run from the original folder. 

Manually disable Apache as a Service:

1.       Type this in your Windows Search:   cmd

2.       Right click the Command Prompt App and select "Run as Administrator".

3.       Now type this in the command prompt:

  sc delete ApacheHTTPServer<enter>

When you start your new grid, Apache will then re-install itself.
