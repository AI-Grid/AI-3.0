Smart Start

 

Smart Start regions are shut down when no one is in them.   A Smart Start region appears as a dual blue arrow ’recycle’ icon in the Regions panel:

Smart start work by deteing teleports.  When someone teleports into the system, or logs in, the Grid will determine if their destination ios running.  If the region is booted. the teleport happens. If the region is not rewady, but enabled, the system send the avatar to a Welcome waiting area.  The region is booted. When the original destination region is ready for logins, the avatar is telepported to the original detination.

Enable Smart Start

Smart tart is disabled by default. You first enable it globaly in Setup->ettings->Regions:

 

This can be enabled for any region.  Edit any region and set the Smart Start Checkbox to True. This goes into effect immediately.

If the region is running and no avatar  in it, it will shut down within one minute.
