SSL Certficates

The SSL setup screen lets you create a free SSL certificate for Apache.  This certificate will automatically renew.

The process is simple and automatic. But there are some rules:

1)       You must be running the Apache webserver.

2)       Apache must be using port 80.  After this step completes you can use port 443 for SSL.

3)       You must be using a domain name in Setup->Settings->HG DNS Name.

4)       The domain must be able to available to the Internet so the system name can be validated.

 

Step 1.  Click “Create Certificate”. The System will take about one minute to create and authorize the certificate.

Step 2.  Stop and Start Apache.  Or click Restart if it is available.
