DEBUG

The Help=> Debug screen contains some debugging tools.  The “True” radio button enables the function, and “False” disables the function. The function occurs when you press Apply is True is set

Benchmark – This saves internal benchmarks to a Benchmark.Log file.  These have time stamps of internal actions.  The one second timer starts the benchmark.

Speak – Uses the Text to Speech API to speak a test message.

Make Voices -  Makes wav and Mp3 files in female or male voices.  You add lines of text to the debug window, set the switch to True, and select Text to Speech in the Pulldown.   Click Apply. DreamGrid will save a wave file using the voice you selected in the Region Panel.  If the line begins with F:  this will use the female voice.  A line beginning with M: will use the male voice.  The files will appear in Apache\htdocs\Stats\TTS as 44.1Khz stereo wav files that can be uploaded, and as MP3 files that can be streamed into Opensim via MOAP. 

Send Alert to a User – Prompts for an Avatar name and will then send an alert to that person, wherever they are.

Teleport API – Prompts for an Avatar name, locates the person, and send them to a region. Part of Smart Boot debugging

Debug (off, 1 Minute …..) – The system prints debug stats on each region every hour. You can speed this up or slow it down. Requires a region restart. The new setting is saved and will be used in the future.
