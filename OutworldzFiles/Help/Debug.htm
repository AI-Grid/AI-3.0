DEBUG

The Help=> Debug screen has debugging tools.  The “True” radio button enables the function, and “False” disables the function. The function occurs when you press Apply is True is set

Teleport API – Prompts for an Avatar name, locates the person, and send them to a region. Part of Smart Boot debugging effort.

Debug (off, 1 Minute …..) – The system prints debug stats on each region every hour. You can speed this up or slow it down. Requires a region restart. The new setting is saved and is sticky across reboots.  Use it to get rapid glances at Opensim region performance.

Debug LandMaker – Requires that the Smart Boot and Endless land and sea be enabled.  Creates multiple avatars that go to and from regions to create a large area of auto generated land. USE WITH CAUTION, for debug only!
